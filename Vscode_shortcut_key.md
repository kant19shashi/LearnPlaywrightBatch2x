# VS Code Keyboard Shortcuts

## File Operations
| Shortcut | Action |
|---|---|
| `Ctrl + S` | Save file |
| `Ctrl + Shift + S` | Save all files |
| `Ctrl + P` | Open file by name |
| `Ctrl + W` | Close current editor |
| `Ctrl + Shift + T` | Reopen closed editor |

## Editing
| Shortcut | Action |
|---|---|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + /` | Toggle line comment |
| `Shift + Alt + A` | Toggle block comment |
| `Alt + Up` | Move line up |
| `Alt + Down` | Move line down |
| `Shift + Alt + F` | Format document |
| `Ctrl + D` | Select word (multi-select) |
| `Ctrl + L` | Select entire line |

## Search & Replace
| Shortcut | Action |
|---|---|
| `Ctrl + F` | Find |
| `Ctrl + H` | Replace |
| `Ctrl + G` | Go to line |

## Navigation
| Shortcut | Action |
|---|---|
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + Shift + O` | Go to symbol |
| `F2` | Rename symbol |
| `F8` | Next error |
| `Shift + F8` | Previous error |

## UI
| Shortcut | Action |
|---|---|
| `Ctrl + B` | Toggle sidebar |
| `Ctrl + ,` | Open Settings |
| `Ctrl + \` | Split editor |
| `Ctrl + 1 / 2 / 3` | Focus editor group |
| `Ctrl + Tab` | Switch open editors |
| `Ctrl + ` ` | Open terminal |
| `Ctrl + Shift + ` ` | New terminal |
| `Ctrl + K Z` | Toggle Zen Mode |
